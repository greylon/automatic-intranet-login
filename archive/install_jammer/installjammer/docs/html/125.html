1.2.5 (February 5th, 2008)


This release contains minor bug fixes and enhancements.



Minor Bug Fixes

  • Open and Save project files with a utf-8 encoding so that non-ASCII text is automatically translated and made correct with each load and save.
  • Fixed a bug where copying and pasting and object would miss the text properties.
  • Fixed a bug in the Windows Service Test condition where the condition would error-out if the service did not exist.
  • Fixed a bug in the Modify Widget action that didn't handle framed widgets.
  • Fixed a bug in the Focus on Widget action where it didn't handle widgets embedded in frames.



Minor Enhancements

  • Added new Size property to Components.
  • Added new Size property to File Groups.
  • Added new Display Name property to Components.
  • Added new Display Name property to Setup Types.
  • Added a new Encoding property to the Read File Into Virtual Text action.
  • Added new Register Windows Library action.
  • Added new Unregister Windows Library action.
