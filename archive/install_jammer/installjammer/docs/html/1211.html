1.2.11 (December 1st, 2008)


This release contains minor bug fixes and enhancements.



Minor Bug Fixes

  • Fix a bug in the uninstaller that can occur on Windows systems where the .vbs file extension is messed up in the registry.
  • Fixed a bug in the Uninstall Selected Files action that would sometimes occur when using the Remove from Uninstall action.
  • Fixed a bug that would only occur on some Windows system where a permission denied error would appear on startup.
  • Fixed a bug where <%Platform%> was not expanded in the name of archive files.
  • Fixed a bug where passing /help as an argument to an installer on Windows would show a message box with the title "In-Valid Argument."
  • Fixed a bug where double-clicking a check box in an option tree would cause the check button to toggle twice, but the command wouldn't fire.  This leaves some components in the wrong state.
  • Fixed a bug where going back-and-forth between a custom setup component screen could leave the components in an unknown state.


Minor Enhancements

  • Add Directory to Path and Add Environment Variable will now also make the same change to the running installer's environment.
