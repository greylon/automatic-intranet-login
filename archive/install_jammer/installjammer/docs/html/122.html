1.2.2 (October 25th, 2007)


This is a minor bug fix release.




Minor Bug Fixes

  • Fixed a bug on UNIX platforms where creating a directory that isn't writable would cause the rest of the install to fail when it tried to write into that directory.  Directory permissions are now set after all files have been installed.
  • Fixed a typo in Uninstall Selected Files that would cause a failure when restoring a registry value on Windows. [Bug #1819503]
  • Fixed a typo in ::InstallAPI::EncodeURL that would cause Launch Web Browser actions to fail. [Bug #1819487]
