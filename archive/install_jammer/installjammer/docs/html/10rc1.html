1.0rc1 (May 10th, 2006)


Bugs

  • Fixed a bug that was causing installers to not all share the same background color.  This would cause the installers to look odd on most UNIX platforms because the colors didn't match.
  • Fixed a bug where the radiobuttons on the license pane were not reflecting state properly.
  • Fixed a bug that was causing InstallJammer to ignore the Allow Language Selection property.
  • Fixed a bug in the install builder that would cause the builder not to remember paned window locations between sessions.
  • Fixed a bug in the Execute External Program action that would cause programs to fail if they had spaces or backspaces in their names.


Enhancements

  • Execute Script actions now execute at the global level.
  • Added a Working Directory property to the Execute External Program action.
