1.2.1 (October 23rd, 2007)


This release contains minor bug fixes and enhancements.



A note on command-line changes


The command-line parsing was changed in the 1.2 release, and this has caused a bit of pain for some users.  I would recommend reading the help page on the command-line, but the long and short of it is that if you are building from the command-line, the safest bet is to make --build the last switch before your project file name.  Some new command-line options were added to this release, and I urge you to read the documentation to learn their uses, but the note about --build being last is important.  It's not required, but you're going to get into a lot less trouble if you heed my advice. :)



Minor Enhancements

  • Added an official exception to the GPL license for installers.  See License for details.
  • Added new Rename Project option for projects.
  • Added new Duplicate Project option for projects.
  • Made Delete Project a lot safer and made the message dialogs a clearer as to what is about to be deleted.
  • Removed the Save as option from the File menu.  It only caused confusion.  Use Duplicate Project instead.
  • Made the New option on the File menu create a new project with all the defaults.
  • Added a new New Project Wizard option under the File menu to create a new project with the existing wizard.
  • Added some debug output to the uninstall process and cleanup.


Minor Bug Fixes

  • Updated the FreeBSD binaries.  They were missed in the last release.
  • Fixed a bug that would cause command-line builds to fail if they discovered a new file before building.
  • Fixed a bug where the Target Filename property on files was being ignored.
  • Fixed a bug where a project that needed conversion that was built from the command-line would fail with an error.
  • Fixed a typo that caused the Add Windows Registry Key action to disappear from the last release.
  • Fixed a bug where an Add to Uninstall action that added a directory to be forcefully removed wasn't actually being forcefully removed.
  • Fixed a missing option bug in Launch Web Browser.
