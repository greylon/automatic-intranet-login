1.0 (June 18th, 2006)


As of the 1.0 release, InstallJammer is now released under the GNU General Public License (GPL).



Bugs

  • Fixed a bug that would cause user input into an entry field to be substituted as a boolean (0 or 1) value as they are typing.
  • Fixed a bug that would cause files that have been installed previously to fail when trying to install over them.
  • Fixed a bug that would cause the Make New Folder button on the Choose Directory dialog to fail.
  • Fixed a bug that would cause the virtual text table to error when switching between languages if an edit was currently active.
  • Fixed a bug that was causing the result virtual text not to be set for an Execute Script action.
